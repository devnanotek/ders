Teorik Bilgi Bankası (Tam Müfredat)

Bilgisayarın gelişimi 4 temel döneme ayrılır:

  • 1. Nesil (1940-1956): Vakum Tüpleri kullanıldı. ENIAC bu dönemin en ünlü devasa bilgisayarıdır. Çok ısınır ve koca bir odayı kaplardı.
  • 2. Nesil (1956-1963): Transistörler icat edildi. Bilgisayarlar küçüldü ve hızlandı.
  • 3. Nesil (1964-1971): Entegre Devreler (IC) kullanılmaya başlandı. Klavye ve monitör devreye girdi.
  • 4. Nesil (1971-Günümüz): Mikroişlemciler (CPU) dönemi. Kişisel bilgisayarların (PC) doğuşu.

Önemli İsimler:

  • Charles Babbage: Bilgisayarın babası olarak bilinir. "Fark Makinesi"ni tasarlamıştır.
  • Ada Lovelace: Dünyanın ilk bilgisayar programcısıdır. Babbage'ın makinesi için ilk algoritmayı yazmıştır.

Donanım (Hardware): Bilgisayarı oluşturan fiziksel parçalardır. Dokunabiliriz.

  • Örnekler: Anakart, İşlemci (CPU), RAM, Sabit Disk (SSD/HDD), Monitör, Klavye.

Yazılım (Software): Donanımı yöneten ve işlem yapmasını sağlayan kod topluluğudur.

  • Sistem Yazılımı: İşletim sistemleri (Windows, Linux, MacOS).
  • Uygulama Yazılımı: Photoshop, Excel, Oyunlar, Web tarayıcıları.

Bilgisayarın çalışma döngüsü 3 ana adımdan oluşur:

  1. Giriş (Input): Kullanıcıdan veri alınır. (Örn: Klavye üzerinden sayı girmek, fareyle tıklamak).
  2. İşlem (Process): Merkezi İşlem Birimi (CPU), gelen veriyi talimatlara göre işler. Matematiksel hesaplamaları ALU (Aritmetik Mantık Birimi) yapar.
  3. Çıkış (Output): İşlenen sonuç kullanıcıya gösterilir. (Örn: Ekrandaki görüntü, yazıcıdan çıkan kağıt).

İkilik (Binary): Sadece 0 ve 1 rakamlarından oluşur. Bilgisayarın ana dilidir.

Onluk (Decimal): Bizim kullandığımız 0-9 arası rakamlardır.

Birimler:

  • 1 Bit = 0 veya 1
  • 8 Bit = 1 Byte
  • 1024 Byte = 1 KB
  • 1024 KB = 1 MB

Dönüşüm mantığı: Onluk sayıyı sürekli 2'ye böleriz ve kalanları tersten yazarız.

Algoritma, bir problemin çözüm adımlarıdır. Temel özellikleri:

  • Başlangıcı ve Sonu olmalı: (Sonluluk)
  • Net olmalı: Her adım açıkça anlaşılmalıdır.
  • Etkin olmalı: Gereksiz adımlar içermemelidir.

Akış Diyagramı Şekilleri:

  • Elips: Başla / Bitir
  • Paralelkenar: Veri Girişi (Input)
  • Dikdörtgen: İşlem / Atama
  • Eşkenar Dörtgen (Baklava): Karar Yapısı (If)

  • Editör: Kod yazdığımız metin düzenleyicidir (Notepad++, VS Code).
  • Derleyici (Compiler): Kodun tamamını bir kerede makine diline çevirir ve bir çıktı dosyası (EXE) üretir. (Örn: C# Derleyicisi)
  • Yorumlayıcı (Interpreter): Kodu satır satır okur ve o an çalıştırır. (Örn: Python)
  • Bağlayıcı (Linker): Derlenen küçük kod parçalarını birleştirip tek bir program haline getirir.

  • Syntax Error (Sözdizimi Hatası): Yazım kuralı hatasıdır. Program asla çalışmaz. (Örn: Noktalı virgül unutmak).
  • Run-time Error (Çalışma Zamanı Hatası): Program çalışırken ortaya çıkan beklenmedik durumlardır. (Örn: Bir sayıyı sıfıra bölmeye çalışmak).
  • Logic Error (Mantıksal Hata): Program çalışır ama sonuç yanlıştır. Bulması en zor hatadır. (Örn: İki sayıyı toplayacakken yanlışlıkla çarpmak).

Değişken Nedir? Bilgisayar belleğinde (RAM) verilerin geçici olarak saklandığı isimlendirilmiş alanlardır.

Veri Tipleri:

  • Tam Sayılar: byte, short, int, long.
  • Ondalıklı Sayılar: float, double, decimal.
  • Metinsel: char (tek harf), string (kelime/cümle).
  • Mantıksal: bool (true/false).

Adlandırma Kuralları: Sayı ile başlayamaz, boşluk içeremez, özel karakter barındıramaz (alt tire _ hariç).

Temel Kod Sözlüğü

int:

Tam sayıları tutar. Örn: 1, 50, -42.

double / float:

Ondalıklı (küsuratlı) sayıları tutar. Örn: 3.14, 1.5.

string:

Metinleri tutar. Çift tırnak içinde yazılır. Örn: "Antigravity"

bool:

Sadece iki değer alır: True (Doğru) veya False (Yanlış).

Console.WriteLine():

Verilen değeri ekrana yazar ve alt satıra geçer.

Console.ReadLine():

Kullanıcıdan bir veri (yazı) girilmesini bekler.

Convert.ToInt32():

Gelen metni bir sayıya dönüştürür. Matematik için şarttır.

for döngüsü:

Belirli bir sayıda işlem yapmak için kullanılır.

Dört İşlem Uygulaması

NE YAPIYOR?
Kullanıcıdan iki sayı alır, sonra 1-4 arası bir rakam seçmesini ister. Seçilen rakama göre (+, -, *, /) yapar. 
using System;
namespace dortislem {
    class Program {
        static void Main(string[] args) {
            Console.WriteLine("Birinci sayıyı girin");
            int sayi1 = Convert.ToInt16(Console.ReadLine()); // Convert ile sayıya çeviriyoruz
            Console.WriteLine("İkinci sayıyı girin");
            int sayi2 = Convert.ToInt16(Console.ReadLine());
 
            Console.WriteLine("Toplama için --> 1\nÇıkarma için --> 2\nÇarpma için --> 3\nBölme için --> 4");
            int islem = Convert.ToInt16(Console.ReadLine());

            if (islem == 1) Console.WriteLine("Sonuç: " + (sayi1 + sayi2));
            else if (islem == 2) Console.WriteLine("Sonuç: " + (sayi1 - sayi2));
            else if (islem == 3) Console.WriteLine("Sonuç: " + (sayi1 * sayi2));
            else Console.WriteLine("Sonuç: " + (sayi1 / sayi2));
            Console.ReadLine();
        }
    }
}

5'e Bölünen & 7'ye Bölünemeyenler

NE YAPIYOR?
1'den 1000'e kadar olan sayıları tarar. Eğer sayı 5'e kalansız bölünüyor VE 7'ye tam bölünmüyorsa onu ekrana yazar. Sonunda kaç adet olduğunu ve toplamını söyler. 
int toplam = 0, adet = 0;
for (int i = 1; i <= 1000; i++) {
    // && 've' demektir. Her iki şart da sağlanmalı.
    if ((i % 5 == 0) && (i % 7 != 0)) {
        Console.WriteLine(i);
        toplam += i; // Sayıyı toplama ekle
        adet++;      // Adeti bir artır
    }
}
Console.WriteLine("Toplam: " + toplam + " | Adet: " + adet);

Hafta 3: Gelişmiş Döngü ve Metot

Restoran Menü Uygulaması

NE YAPIYOR?
"Metotlar" (static void) kullanarak ana menüyü parçalara böler. Yemek, içecek veya tatlı seçtiğinde bakiye (hesap) sürekli güncellenir. 
static int hesap = 0;
// Ana Döngü
static void Main() {
    int secim;
    do {
        Console.WriteLine("1-Yiyecek, 2-İçecek, 3-Hesap Öde");
        secim = Convert.ToInt32(Console.ReadLine());
        if (secim == 1) yiyecekler();
        else if (secim == 2) icecekler();
    } while (secim != 3);
    Console.WriteLine("Toplam Borç: " + hesap + " TL");
}

static void yiyecekler() {
    Console.WriteLine("1-Et(20), 2-Mantı(15)");
    int s = Convert.ToInt32(Console.ReadLine());
    hesap += (s == 1) ? 20 : 15;
}

Sayı Tahmin Oyunu v2

NE YAPIYOR?
Bilgisayar `Random` sınıfı ile 1-100 arası bir sayı tutar. Kullanıcının her tahmininde "Yukarı" veya "Aşağı" diyerek yardım eder. Sonunda kaç kerede bildiğini söyler. 
Random rnd = new Random();
int tutulan = rnd.Next(1, 101); // 1-100 arası
int tahmin = 0, sayac = 0;

Console.WriteLine("Aklımda bir sayı var, bil bakalım!");
do {
    tahmin = Convert.ToInt16(Console.ReadLine());
    sayac++;
    if (tahmin > tutulan) Console.WriteLine("Daha Aşağı!");
    else if (tahmin < tutulan) Console.WriteLine("Daha Yukarı!");
} while (tahmin != tutulan);

Console.WriteLine("Tebrikler! " + sayac + ". tahminde bildin.");

Negatif Sayı Girince Sonlanan Döngü

ANALİZ:
`while(true)` sonsuz döngüsü kullanılır. Kullanıcı her sayı girdiğinde `if` ile kontrol edilir. Eğer girdiği sayı 0'dan küçükse (negatifse) `break` komutu ile döngü kırılır ve program biter. 
while (true) {
    Console.WriteLine("Bir sayı giriniz (Çıkmak için negatif bir sayı):");
    int sayi = Convert.ToInt32(Console.ReadLine());
    if (sayi < 0) {
        Console.WriteLine("Negatif sayı girildi, program sonlandırılıyor.");
        break; 
    }
    Console.WriteLine("Girdiğiniz sayı: " + sayi);
}
