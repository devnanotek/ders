Soru 1 ID kullanarak stil oluşturma
HTML 
<h1 id="baslik1">Başlık 1</h1>
<h2 id="baslik2">Başlık 2</h2>
<p id="icerik">Bu bir içerik paragrafıdır. ID kullanarak stillendirme yapıyoruz.</p>
CSS 
#baslik1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

#baslik2 {
    color: red;
    font-size: 20px;
    text-align: center;
}

#icerik {
    color: green;
    font-size: 16px;
    text-align: justify;
    padding: 20px;
}
Önizleme:

Başlık 1

Başlık 2

Bu bir içerik paragrafıdır. ID kullanarak stillendirme yapıyoruz. Her ID sadece bir kez kullanılabilir ve benzersiz olmalıdır.

    </div>
    <div class="kutu2">
        <h3>İkinci Kutu</h3>
        <p>Bu sarı renkli bir kutudur.</p>
    </div>

        <h3>Üçüncü Kutu</h3>
        <p>Bu kırmızı renkli bir kutudur.</p>

    width: 100%;
    border: 2px solid black;

}
